webapi angular project
this a simple project made by Asp.net core Webapi and angular

frontend:
first page made by THREEJS and some buttons to link to other pages
page of Stadium Cups Articles Legends use GET METHODE and get datas from backend
there is a login and register page that use Jwt from backend
there is an admin page named AdminEdit which it shows the datas of the tables of database in a table with DELETE function from database
there is AOS library used in contact page which make a animation when scroll
there is a loader in main pages and it run till the api is fetched
there is 404 page


backend

used Entity framework for linking to database used Dapper for linking to database used Fluent Validation for validate the incomes used Jwt for login and register used swagger for checking api


database is Sql server
